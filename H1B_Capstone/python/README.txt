This folder contains python code files created for the Capstone

Files:

analysis.py - final full-model analysis script

analysis_tsne.py - script to generate t-SNE plot

county_pop.py - script used to call the Census Population Estimate API in order to obtain county population statistics.

cross_valitation.py - script used for performing cross valitation to determine full model to run

fcc_census.py - script used to call the FCC Area API in order to obtain census block information and FIPS codes.

h1b_census_merge.py - script combining cleaned h1b kaggle data and cleaned fcc/census data

h1b_cleanup.py - script used to clean the original Kaggle dataset.

This folder contains Jupyter notebooks for some of the visualization code. These are adaptations of code in the python folder

Files:

Average_Images.ipynb - adaptation of the avg_image.py script, visualizing the average image from each category.

Color_Channel_Plots.ipynb - adaptation of the Image_EDA.py script, generate plots of color value frequency for all photos, testing photos and validation photos.

Model_Predictions.ipnb - adaptation of Visualize_Models.py, visualizes the predicted values for 10 photos for each of my models.
